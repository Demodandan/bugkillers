## 数据库表设计  v1.0

####用户表  user
* id    用户主键
* name       用户名
* nickname   昵称
* password   密码
* email      邮箱
* dr         删除标示位
* registtime 注册时间
* logintime  最近登录时间
* headpic    头像
* bk_create  创建时间
* bk_modified 修改时间


####问题表   question

* id  问题主键
* pk_tag          问题标签
* createuser      提问人pk
* createtime   提问时间
* updatetime   更新时间
* updateuser   更新人pk
* lookedtimes  被查看次数
* dr           删除标示位
* text         问题正文
* observers    关注者
* bk_create  创建时间
* bk_modified 修改时间

####标签表   tag

* id      标签主键
* tagname     标签名称
* bk_create  创建时间
* bk_modified 修改时间


####评论表  comment

* id  评论主键
* pk_question 所属问题pk
* text        评论内容
* commetuser  评论人
* commenttime 评论时间
* order       评论顺序
* dr          删除标示位
* bk_create  创建时间
* bk_modified 修改时间
